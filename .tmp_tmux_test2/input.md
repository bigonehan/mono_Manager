# tmux_split
- tmux 세션/패널 생성은 비대화형 명령으로 수행한다.
- tmux 세션/패널 이름 충돌을 방지하기 위해 고유 식별자를 사용한다.
> 고유 식별자를 포함한 tmux 세션을 생성한다.
> 필요한 split 패널 구성을 순서대로 완료한다.

# school_landing_page
- 결과물은 html, css, javascript 정적 웹 자산 구조를 유지한다.
- 빌드 도구나 추가 프레임워크를 도입하지 않는다.
> html 파일을 생성 또는 갱신한다.
> css와 javascript 파일을 생성 또는 갱신한다.

# tmux_bash
- 검증 명령은 bash 기준 문법만 사용한다.
- 검증 순서는 재현 가능하도록 고정한다.
> 각 tmux 패널에서 지정된 bash 검증 명령을 순차 실행한다.
> 패널 실행 상태를 확인해 실행 단계 완료 여부를 기록한다.
